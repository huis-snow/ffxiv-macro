우주개척 매크로 관리 사용법
1. 시작하기
📁 데이터 가져오기
  • 파일 업로드: 상단 "데이터 가져오기" 버튼 클릭
  • JSON 붙여넣기: 우하단 파란색 버튼 () 클릭
  • 기존 매크로 데이터가 있다면 자동으로 목록에 표시됩니다
💾 데이터 저장
  • 브라우저에 자동으로 캐시됩니다
  • "데이터 내보내기"로 JSON 파일로 백업 가능
2. 매크로 조회
🔍 검색하기
  • 작업량 검색: 원하는 작업량 입력 후 "검색" 버튼
  • 전체 보기: "전체" 버튼으로 모든 매크로 표시
  • 정렬: 테이블 헤더 클릭으로 정렬 (작업량, 품질, 내구도)
📋 매크로 선택
  • [선택] 버튼 또는 테이블 행 클릭
  • 선택하면 하이라이트 및 첫 블록 자동 복사
  • 우측 패널에서 상세 정보와 블록별 복사 가능
3. 매크로 생성
📝 정보 입력
  • 작업량, 최대품질, 초기품질, 내구도: 필수 정보
  • 음식, 메모: 선택사항
  • 매크로 내용: 하단 텍스트 영역에 입력
💾 저장 및 관리
  • 저장: 새 매크로 저장
  • 수정: 조회/열기로 기존 매크로 불러와서 수정
  • 초기화: 폼 내용 초기화
4. 고급 기능
⚡ 빠른 생성
  • 검색 시 작업량 입력 후 "매크로 생성" 버튼 표시
  • 클릭하면 생성 탭으로 이동하며 작업량 자동 입력
💾 캐시 관리
  • 수동 저장: 즉시 브라우저에 저장
  • 캐시 정보: 저장된 데이터 개수 확인
  • 캐시 삭제: 모든 로컬 데이터 삭제
📝 블록 복사
  • 매크로는 15줄씩 블록으로 나뉩니다
  • 각 블록별로 개별 복사 가능
  • 매크로 선택 시 첫 블록 자동 복사
💡 팁과 주의사항
✅ 좋은 사용 습관
  • 정기적으로 "데이터 내보내기"로 백업하세요
  • 매크로에 의미있는 메모를 추가하세요
  • 음식 정보를 정확히 입력하세요
  • 테이블 헤더 클릭으로 원하는 정렬 방식 사용
⚠️ 주의사항
  • 브라우저 데이터 삭제 시 캐시가 사라질 수 있습니다
  • 같은 조건의 매크로는 덮어쓰기됩니다
  • 매크로 내용은 정확한 게임 명령어를 사용하세요
  • 최대품질은 정수값만 입력 가능합니다
